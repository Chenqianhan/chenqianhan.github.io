训练细节

Posted on 2018-09-01

It’s wrong that ConvNets need a lot of data to train.
事实上少量的数据同样可以，一般来说很少完全地开始进行卷积神经网络的训练，通常会先进行预训练和细调。

###预训练
可以先在一些大的数据集训练如ImageNet data，再去在我们所感兴趣的数据中训练。
首先得到ImageNet进行预训练的网络，再将最上方的层（分类器）移除，然后将这个卷积神经网络看成是一个固定特征提取器来进行训练，把这个特征提取器置于你的数据集上方，然后将替换原先作为分类器的层。

再根据你的数据集大小来确定如何对卷积网络的最后一层进行训练或者对整个网络的一部分反向传播进行微调。如果有更多的数据，则可以在整个网络进行更深的反向传播。

caffe model zoo已经有很多预训练的神经网络模型。
